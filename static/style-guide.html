I am hoping to include all HTML elements on this page, which will make it long, but hopefully not too complicated. This is not bells and whistles, it just every element so you can put your basic styles in, and have something to start with.

Aenean ultricies mi vitae est. This is how a strong tag would look and this is using the b tag, don't get them confused. Strong is for important text, b doesn't have any semantic value. This is a b tag with a class of lede, for extra versatility (ie first sentence of a paragraph). Sit amet est et sapien ullamcorper pharetra. This is how an em tag would look. Again don't muddle this up with i which you can use but has no semantic value. Aenean fermentum, elit eget tincidunt condimentum.
Did you notice how I just used a br. This is for breaking a line inside a paragraph or title NOT for clearing elements. Close and reopen paragraphs for new paragraphs.

I am pretty awesome, I can use del.. I am going to try and define the dfn element. I should wrapped dfn tag in dfn. Confused much? I am going to insert some text here using the ins tag.
